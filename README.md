# 🌿 GreenQuest
GreenQuest, karbon ayak izinizi takip etmenizi sağlayan oyunlaştırılmış bir platformdur.

## 🚀 Teknolojiler
- **Frontend:** React, Tailwind CSS, Chart.js
- **Backend:** Node.js, Express
- **Database:** MySQL
- **CI/CD:** GitHub Actions

## 🛠️ Kurulum
1. `database.sql` dosyasını MySQL'de çalıştırın.
2. `server` ve `frontend` klasörlerinde `npm install` komutunu çalıştırın.
3. Projeyi `npm run dev` ve `node index.js` ile başlatın.
