# 🌿 GreenQuest

GreenQuest is a gamified platform that helps users track and reduce their carbon footprint through daily activities and challenges.

---

## 🚀 Features

* User authentication (register/login)
* Carbon footprint tracking
* Gamified challenges and rewards
* Progress dashboard
* Environmental impact insights

---

## 🛠️ Technologies Used

### Frontend

* React.js
* HTML, CSS, JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQL (MySQL / PostgreSQL)

### DevOps

* GitHub Actions (CI/CD)

---

## ⚙️ Installation & Running the Project

### 1. Clone the repository

```bash
git clone https://github.com/Yasin05-code/carboncalculator.git
cd carboncalculator
```

### 2. Run Backend

```bash
cd server
npm install
npm start
```

### 3. Run Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🔄 CI/CD Pipeline

This project uses **GitHub Actions** for continuous integration and deployment.

Pipeline includes:

* Installing dependencies
* Running build process
* Running basic tests
* Ensuring code integrity on every push

Workflow file is located in:

```
https://github.com/Yasin05-code/carboncalculator/blob/main/.github/workflows/main.yml
```

## ⚠️ Risk Management
